# chie Cafe

## 作品名
Chie Cafe（チエ カフェ）

## 作品の説明
Chie Cafe は、コーヒーを楽しみながら仕事や勉強ができる  
コミュニティ型コワーキングカフェをイメージして作成した Web ページです。  
HTML で構造を作成し、CSS でデザインを整え、JavaScript でカート機能などの動きを実装しました。

## 工夫した点・見どころ
- コーヒーメニューやコーヒー豆を「カートに追加」できる機能を JavaScript で実装しました  
- ミルク変更による料金加算など、実際のカフェを意識した仕組みを取り入れました  
- コーヒー豆は CSS の flexbox を使い、3つ横並びで表示しています  
- 写真を多く使い、見やすく親しみやすいデザインを意識しました  

## 使用技術
- HTML  
- CSS（Flexbox）  
- JavaScript
